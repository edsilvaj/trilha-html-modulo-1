Introdução ao HTML

No HTML existe uma estrtutura que deve ser seguida para a confecção de um documento, ou seja, o documento .html deve conter cabeçalho, tipo de documento, e corpo (que aparecerá de fato para quem visualiza o site).

Essa estrutura será explicada na seção de tags essenciais.

Toda tag deve ser iniciada com <"tag"> e finalizada com < /"tag">

Tags Essenciais

Voltar
Como tags essenciais, o HTML possui algumas que precisam estar em um código de forma obrigatória a fim de declarar o tipo do documento, o cabeçalho, e o corpo do site.
São elas:
  • < !DOCTYPE html > | Define o tipo de documento para HTML. |
  • < head > | Que é o que define o cabeçalho do site, ou o que será usado como estilos etc.|
  • < title > | É a tag que define o nome na aba. |
  • < body > | É o que vai definir onde fica o corpo do site ou texto.

Tags de Texto

Voltar

  • Tag H

    • A tag H, serve para marcar títulos, e possuí 6 (seis) tipos, ou seja, vai desde o h1 até o h6.

  • Tag P

    • A tag P, serve para indicar um texto, ou seja, um parágrafo e deve ser usada quando será escrito algo para que seja feita de forma continua e não faça parte de um título.

  • Tag blockquote

    • A tag bloackquote, serve para ser usadas em citações, ou seja, deixar o texto formatado dentro de um parágrafo em evidência e separado do parágrafo em questão.

  • Tag strong

    • A tag strong, evidencia um texto, ou seja, marca ele em negrito. Dessa forma, o texto fica em evidência sem ser alterada a formação de todo o parágrafo, pois essa tag pode ser usada de forma isolada em palavras ou trechos do texto.

  • Tag U

    • A tag U, serve para deixar um texto sublinhado.

  • Tag I

    • A tag I, serve para deixar um texto em itálico.

  • Tag sup

    • A tag sup, deixa um tipo de marcador na parte superior, ou transfere um texto para a parte de cima do parágrafo.

  • Tag mark

    • A tag mark, torna o texto marcado em amarelo.

Lista ordenada e não ordenada

Voltar
Com o HTML podemos fazer listas, ordenadas e não ordenadas*
*A lista ordenada pode ter sua forma alterada com CSS

Lista ordenada

A Lista ordenada, é uma lista que contém índice, ou seja, ordem e portanto é numerada de 1 a N, sendo N a quantidade de itens dessa lista.
Para que seja realizada a lista ordenada, deve se dar início a ela com a tag < ol > e listar os itens com a tag < li >.

Lista não ordenada

A Lista não ordenada, é uma lista que não contém índice, ou seja, ordem e portanto é mostrada sem uma numeração.
Para que seja realizada a lista não ordenada, deve se dar início a ela com a tag < ul > e listar os itens com a tag < li >.

Uso de Links

Voltar
O uso de links, serve para que seja referênciado um link em outra página, ou na mesma página e pode ser implementado da seguinte maneira.
< a href="link" > "texto" < /a >
Sendo o href*, a onde esse "botão" será direcionado.
*O href, pode ser tanto links externos, quanto os documentos da página web que você está construindo, como index.html, about.html etc.
Atributos do link
Um outro atributo do < a > < /a >, é o target=, esse target pode ser referenciado como _self ou _blank*
Sendo o _self responsável por abrir o link na mesma página que você já está navegando.
E o _blank responsável por abrir o link em uma nova janela.
Dessa forma a sintaxe com esse atributo fica o seguinte: < a href="link" target="_blank ou _self" > "texto" < /a >
*Por padrão o HTML executa esse atributo no modo _self caso não seja especificado.
Outro atributo do link, é o title= "texto", que exibe uma tooltip do link/atalho inserido no código.
Deixando a sintaxe da seguinte forma: < a href="link" tittle="tooltip" > "texto" < /a >
